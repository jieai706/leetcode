-----数组-----
P27:删除数组指定的元素 easy
说明：数组中删除操作一般都是双指针交换元素，题目中描述元素的顺序可以改变，这样题目的难度就更低了，使用两个碰撞指针前后遍历数组就行了。一直指向要删除的节点，一个指向不需要删除的节点，交换彼此即可。
但是如何操作后不能改变元素的顺序，那应该怎么办么？可以使用两个同向的并行指针
